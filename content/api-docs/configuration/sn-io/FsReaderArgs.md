## FsReaderArgs


### Configuration example:
``` json
{
  "fsReader": {
    "Path": "_value_"
  }
}
```
### Environment variables example:
```
fsReader__Path="_string_value_"
```
### Properties:
- **Path** (`string`): Source file system folder path.
