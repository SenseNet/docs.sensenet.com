## DeleteApiKeys
- Method: **POST**.

Deletes api keys of the target user.

### Request example:

```
POST /odata.svc/Root/...('targetContent')/DeleteApiKeys
```
The `targetContent` can be User
### Parameters:
There are no parameters.

### Return value:
An empty result.

### Requirements:
- **AllowedRoles**: Administrators
